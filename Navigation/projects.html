Control

Project Description: Placing how to take care of one's mental health at its core, Control is a Java-based application that helps you talk about your thoughts and emotions through writing and denote your feelings for that day. It also provides random quotes to encourage the user and uplift their spirits. All in all, it is made for users to remind themselves that they are indeed in control and can take care of their mental health.

Product Description: A user can create a profile in the system. After, a user can denote a mood for the day with having the option to write an optional journal entry. The user can then go back and relive their days by clicking a date in the calendar. The moods are color-coded for better visuals, which can potentially help some users. The quotes are queried in a collection and randomly generated every time a user refreshes the home page.

Tools Used: Java, Android Studio, Firebase

Achievement: Second Place

Momentum

Project Description: Momentum is a Java-based application that lets a user add habits, denote habit events when a habit is done as planned, and lets the user share to others their current habit progress.

Product Description: With a uniquely identifiable user in the database system, the user can add a habit with details such as title, reason, frequency, and if it is public or private. The user can also denote a habit as completed using a colour-coded calendar and other date functionalities. When a habit is done, a habit event with details such as comment, photograph, and location can be added. By granting or denying follow requests from others, users can share and follow other people's progress through a cohesive visual indicator for each public habit.

Tools Used: Java, Junit and Robotium Testing Frameworks, Android Studio, Firebase, Google API

Money For Time

Project Description: It is an easy-to-use program to keep track of the times you have spent doing something important. To reward you for your efforts, it calculates how much money you will receive based on your pre-determined inputs.

Product Description: A user can enter a set time (hours and/or minutes) to their profile. Using the entered time, the program calculates how much money they can have for their total time. Data gathered is also automatically saved to a local SQL database.

Tools used: Python, SQL
